How to change the Background Color of a Form
Difficulty: EASY

In this tutorial, I will show you how to change the background color of a form.

1. Select your form by single-clicking on it.
2. Go down to the properties box and find BackColor.
3. Click on the arrow, Click on Web, and select the color of your choice.


You can also use the code way: 
public Form1() {
	InitializeComponent();
	this.BackColor = Color.Red;
}
