How to make a Form Always-On-Top
Difficulty: EASY

In this tutorial, I will show you how to make a form always on top.

1. Select your form by single-clicking on it.
2. Go down to the properties box and find TopMost.
3. Click on the arrow, and select True.


You can also use the code way: 
public Form1() {
	InitializeComponent();
	this.TopMost = true;
}
